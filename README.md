# Advanced Calculator

A sleek, responsive, and functional web-based calculator built with HTML and CSS. This project demonstrates a clean user interface design and simple JavaScript-based arithmetic logic.

## 🚀 Features

- **Basic Arithmetic**: Supports addition, subtraction, multiplication, and division.
- **Decimal Support**: Perform calculations with floating-point numbers.
- **Clear Functionality**: Quickly reset the display with the 'C' button.
- **Responsive Design**: Uses CSS Grid for a perfect layout on any screen size.
- **Modern UI**: Features a clean aesthetic with hover effects and distinct operator buttons.

## 🛠️ Technologies Used

- **HTML5**: For the structure and semantic markup.
- **CSS3**: For the layout (Grid), styling, and interactive hover effects.
- **JavaScript**: Minimal inline logic using `eval()` for rapid calculation.
